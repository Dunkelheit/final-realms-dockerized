# Final Realms, dockerized

Dockerized the [MudRen/FinalRealms](https://github.com/MudRen/FinalRealms) 
repository that contains a barebones version of the Final Realms library.

## Instructions

```
> docker-compose up -d
> telnet localhost 4001
```

Log in with the username `god` and password `god`.

Have fun!
